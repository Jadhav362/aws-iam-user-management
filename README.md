# AWS IAM User Management Project

## Project Overview
This project demonstrates how to create and manage IAM users in AWS.

## Services Used
- AWS IAM
- AWS Management Console

## Steps
1. Login to AWS Console
2. Open IAM service
3. Go to Users
4. Click Create User
5. Enable console access
6. Attach AdministratorAccess policy
7. Create user and save credentials
